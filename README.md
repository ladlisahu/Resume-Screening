## ⚙️ Setup Instructions

### 1. Clone the repo

```bash
git clone <your-repo-url>
cd Resume-Screening
```

### 2. Set up environment variables
Create a .env file and add your Hugging Face token:
```bash 
HF_TOKEN=your_huggingface_token_here
```

## 3. Run the App :
```bash 
streamlit run streamlit_ui.py
```
